Prime Path

Prime Path

POJ 3126

这道题也可以算是到水题吧。搜索加上求素数，这里用的是线性筛。

题目来源: vjudge | POJ

分析

题目要求很简单，给定一个起点和终点，要求用最短的步数到达。每改变一个数算作一步，并且在改变后要求仍然为素数。所以先算出所有素数，然后进行暴力BFS即可。
